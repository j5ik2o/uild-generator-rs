# ulid-generator-rs

[![crates.io](https://img.shields.io/crates/v/ulid-generator-rs.svg)](https://crates.io/crates/ulid-generator-rs)

This is a Rust crate for generating ULIDs.

# Usage

Add this to your Cargo.toml:

```toml
[dependencies]
ulid-generator-rs = "0.0.1"
```
